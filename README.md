## 使用说明
1. 所有算法均采用C++实现
2. 代码包含详细注释，方便理解
3. 多数模板已通过题目验证
4. 可根据需要直接复制使用

- ​**全面性**：涵盖ACM/ICPC常见算法
- ​**高效性**：优化过的实现保证运行效率
- ​**可读性**：代码结构清晰，变量命名规范
- ​**模块化**：各算法独立封装，便于调用
